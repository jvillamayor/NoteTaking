# NoteTaking
Syntax highlighting for Sublime Text and TextMate, for taking readable notes.

key characters:

    //          ends with a new line
    /*          ends with '*/'
    *           ends with '˙' (alt + H on OSX)
    ~           ends with a new line
    ˙           ends with '˙' (key is entered with alt + H on OSX)
